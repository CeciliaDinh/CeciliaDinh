## 🚀 Featured Projects

### 🔗 [Hybrid Recommendation System](#) <!-- Thay '#' bằng link repo Github của bạn -->
> *Python · PyTorch · LightGCN · RecBole · Graph Neural Networks · CLIP*

Enhanced LightGCN with multimodal item embeddings — engineered auxiliary features from raw Amazon Reviews 2023 data and designed 4 fusion architectures (additive, concat-MLP, gated, weighted). Achieved **13% improvement in Recall & NDCG@10** over baseline. Built async image crawling pipeline and CLIP-based visual feature extraction to extend the model toward multimodal recommendations.

<br/>

### 🔗 [Retail Sales Profitability Analysis](#) <!-- Thay '#' bằng link repo Github của bạn -->
> *Python (Pandas, Scikit-learn) · K-means · RFM Segmentation · Power BI*

Diagnosed a high-revenue, low-margin paradox across 200k+ rows of retail data. Engineered RFM features and re-segmented customers into 9 behavioral clusters — identified deal-hunter cohort (Cluster 0: −21.9% profit margin, 28% avg. discount) as primary driver of margin erosion. Quantified break-even thresholds via cost-benefit analysis and proposed tiered, margin-aware discount caps per segment.

<br/>

### 🔗 [Customer 360 Analytics](#) <!-- Thay '#' bằng link repo Github của bạn -->
> *Python · PySpark (Spark 3.5) · RFM · Churn Analysis · Seaborn*

Built an end-to-end analytics pipeline on 4+ years of e-commerce transaction data — from multi-stage PySpark data cleaning to RFM feature engineering and churn labeling. Surfaced a critical retention bottleneck (34% churn after 1–2 purchases), YoY country-level growth anomalies, and product-level conversion gaps between wishlists and purchases.

<br/>

### 🔗 [Airline Loyalty Program Analysis](#) <!-- Thay '#' bằng link repo Github của bạn -->
> *SQL (Window Functions, CTEs) · Google Sheets · Cohort Analysis*

Audited 100k+ member loyalty program — identified a 6% point redemption rate despite 120% YoY adoption growth, diagnosing a loyalty loop failure. Segmented 3 tiers into distinct behavioral profiles and recommended tier-specific retention levers to accelerate point burn rate.
